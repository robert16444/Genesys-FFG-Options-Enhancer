# Genesys FFG Options Enhancer

**Genesys FFG Options Enhancer** is a quality-of-life module for **Foundry VTT** designed mainly for  
**Genesys RPG** system.
It adds several GM and player tools.

---

##  Features

###  GM Tools
####  Roll Request System
- Request skill rolls directly from selected online players
- Choose:
  - Skill
  - Difficulty
  - Challenge (upgrades)
  - Setback
  - Boost
  - Roll mode (public / private / blind / self)
- Players receive a popup and roll using the **native system roll dialog**
- Fully respects Genesys dice upgrade rules

Accessible from **Token Controls → Request a Roll**

---

###  Player Tools
####  Talent & Spell Search
- Advanced searchable talent/spell browser
- Searches:
  - Item talents
  - Talents embedded in species / specializations
- Live filtering by name and description
- Highlighted search terms
- Expand / collapse descriptions
- Optional hiding of talents without descriptions
- Send talent to chat
- Cast spells / talents:
  - Parses activation cost from description
  - Supports multiple levels (popup selection)
  - Automatically subtracts Force Pool / mana

Accessible from **Token Controls → Talent Search**

---

###  Inventory Tools
####  Item Transfer Between Players
- Right-click items in character inventory
- Send **Armour / Gear / Weapons** to another online player
- Recipient can **Accept / Decline**
- Transfer is handled safely by the GM (no permission issues)

---
